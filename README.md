# Project 011: Check if a String is a Palindrome

## Description
This project asks the user to enter a string and checks if it is a palindrome.  
It teaches beginners about string slicing and comparison.

## Skills Learned
- String slicing  
- String comparison  
- User input  
- Conditional statements (`if-else`)  

## How to Run
1. Open `palindrome_check.ipynb` in Jupyter Notebook.
2. Run all cells.
3. Enter any text to see if it is a palindrome.
